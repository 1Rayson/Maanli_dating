For at få fat i brugerens login oplysninger (brugernavn og password):
maanliUserLogin
- Denne indeholder
    id - INT NOT NULL PRIMARY KEY AUTO_INCREMENT,
    username - VARCHAR(50),
    userPassword - VARCHAR(60),
    userID - INT NOT NULL
Hvor userID refererer til brugerens ID fra maanliUserProfile
